# devforge

**Opt-in per project.** Applies only when `.devforge/project.md` exists at the project root. In any other project, ignore this section and follow that project's own methodology.

In devforge-enabled projects: before any task, invoke `devforge:unified-workflow` — it is the dispatch table that sequences superpowers skills, beads bookkeeping, and the Playwright UI gate.

If superpowers is not installed, install it — this plugin is an orchestrator, not a replacement.
